💡 How It Works

Chrome blocks animated GIF favicons. Our solution: JavaScript dynamically swaps the <link rel="icon"> tag's href with canvas-rendered PNG data URIs multiple times per second. Each frame is drawn mathematically — no external files, no server requests. The result is a silky-smooth animated favicon that works in every modern browser.
