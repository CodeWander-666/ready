💡 How This Works

Chrome doesn't support animated GIF favicons (unlike Firefox which does). Our indirect workaround: We use JavaScript to dynamically swap the <link rel="icon"> tag's href multiple times per second. Each frame is a canvas-generated PNG (via canvas.toDataURL()), creating smooth animation that Chrome happily displays.

✅ No external files needed — all frames are generated in-browser.
✅ No server requests — uses inline data URIs.
✅ Performance-friendly — frames are pre-rendered, only the DOM attribute changes.
